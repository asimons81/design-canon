# Research Templates

This directory contains contract templates for research artifacts.

## Files

- `rule-proposal.json` — Template for Track A and Track C anti-slop proposals
- `copy-proposal.json` — Template for Track B contextual AI-copy proposals
- `source-record.json` — Template for Track E source registry entries
- `audit-record.json` — Template for Track C component audit records
- `fixture-manifest.json` — Template for Track D fixture manifests

## Usage

Each template file documents required fields, optional fields, and validation rules. Do not weaken schema constraints to make invalid research pass.
