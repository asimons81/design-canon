# Clean-Room Research

Design Canon's anti-slop corpus is developed through disciplined, independent research grounded in public sources and original observation.

## Core Principle

The project researches the general architectural problem: *How can AI-generated interfaces avoid repetitive visual defaults, generic copy, deceptive interface states, weak hierarchy, and template-like design?*

All findings come from publicly accessible sources or clearly labeled original observations.

## Prohibited Activities

Researchers must not:

- seek leaked or private copies of proprietary design prompt collections
- reconstruct proprietary documents from screenshots, quotes, fragments, or user reports
- use paywall bypasses or authentication workarounds
- copy proprietary prompts or private design corpora
- claim knowledge of unpublished contents of proprietary systems
- treat public marketing claims as evidence for specific anti-pattern rules
- reproduce large passages from any single source
- invent citations
- claim superiority over untested proprietary systems

## Permitted Research Methods

- examining publicly accessible web interfaces
- reading open design standards and specifications
- surveying academic literature on UI patterns and anti-patterns
- analyzing published design-system documentation
- conducting original observation of recurring AI-generated interface patterns
- labeling personal judgment explicitly as opinion

## Attribution Standards

| Claim Type | Evidence Required |
|---|---|
| Observation-based | Source URL, access date, excerpt under 25 words |
| Research-backed | Academic paper citation or public report |
| Original observation | Labeled "original observation" |
| Hybrid | Source reference plus original analysis |
| General knowledge | No citation required (e.g., "blue text on black background is hard to read") |

## Attestation

Every proposal, source record, and fixture must include a clean-room attestation confirming it was developed without access to proprietary prompt collections or unlicensed design systems.
