Spacing? We Don't Know Her

Every margin and padding value on this page is unique. No two adjacent elements share the same spacing. We deliberately avoided any pattern, token, or system.

Our Chaotic Features

Random Padding

Each card has a different internal spacing. The 31px top padding was chosen for no reason.

Arbitrary Gaps

Grid gaps vary per section. 19px here, 27px there, 51px for the stats. Consistency is boring.

No Token System

Every value is a hardcoded one-off. 7px, 11px, 13px, 15px, 17px, 19px — we use them all.

Margins from Mars

Margins are applied in unpredictable ways. Some elements have margin on one side only.
